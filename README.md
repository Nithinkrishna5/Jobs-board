# Jobs For You

A fresh job platform built with React using browser-loaded React and Babel.

## Preview

Open `index.html` directly in your browser.

## Features

- React-based UI with JSX
- Search and category filters
- Save / apply interaction buttons
- Dark mode toggle
- Responsive layout for desktop and mobile

## Files

- `index.html` — main React shell and library imports
- `styles.css` — styling for layout and components
- `app.jsx` — React application code
